CityParking Chat Widget - Basic Example

This is a basic implementation of the CityParking chat widget using the new data attributes approach. The widget appears as a floating chat button in the bottom-right corner of the page.

⚠️ Demo Note: This example shows the widget structure but won't function without a valid bot endpoint URL. Replace the placeholder URLs with your actual bot endpoints.

Implementation Code (Data Attributes - Recommended):

<script 
    src="https://your-cdn.com/chat.min.js" 
    data-chat-url="https://your-bot-endpoint.com/bot/your-bot-id" 
    data-btn-position="bottom-right"
    data-widget-btn-color="#009648"
    defer>
</script>

Alternative JavaScript Method:

<script src="https://your-cdn.com/chat.min.js"></script>
<script>
  window.initCityparkingWidget({
    chatUrl: 'https://your-bot-endpoint.com/bot/your-bot-id',
    language: 'en',
    theme: {
      primaryColor: '#009648',
      position: 'bottom-right'
    }
  });
</script>

Available Data Attributes:

  • data-chat-url - Bot endpoint URL (required)
  • data-btn-position - Position: 'bottom-left' or 'bottom-right'
  • data-widget-btn-color - Button color (any CSS color)
  • data-language - Widget language: 'en', 'nl', 'fr'

Features Demonstrated:

  • Basic chat functionality
  • AI-powered responses
  • Multi-language support (English)
  • Default green theme (#009648)
  • Bottom-right positioning
  • Contact form integration
  • Data attributes configuration
