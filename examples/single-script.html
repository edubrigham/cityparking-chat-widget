This page demonstrates how to integrate the CityParking Chat Widget with just one script tag using data attributes. No need for separate initialization code!

Implementation

The widget on this page was added with this simple single script tag:

<script src="https://cdn.jsdelivr.net/gh/edubrigham/cityparking-chat-widget@latest/dist/cityparking-chat-widget.min.js" data-webhook-url="https://edubrigham.app.n8n.cloud/webhook/4091fa09-fb9a-4039-9411-7104d213f601" data-language="en" data-primary-color="#009648" data-position="bottom-right"> </script>

Available Data Attributes

  • data-webhook-url - Your n8n webhook URL (required)
  • data-language - Widget language: 'en', 'nl', or 'fr' (optional)
  • data-primary-color - Primary color for the widget (optional)
  • data-position - Widget position: 'bottom-left' or 'bottom-right' (optional)
